# 项目简介

## 核心功能
- 基于YOLOv8的轨道缺陷检测，支持Crack、Putus、Spalling、Squat四类缺陷识别
- 支持多种输入源检测：单张图片、图片文件夹、视频文件、本地摄像头
- 提供完整界面功能：实时显示目标位置、目标总数、置信度等信息
- 支持批量检测与结果查看
- 可自定义检测参数：置信度阈值、IOU阈值
- 支持检测结果保存功能
- 包含1593张标注轨道缺陷数据集

## 适用场景
- 轨道交通安全检测与维护
- 铁路基础设施缺陷自动化识别
- 工业视觉检测系统部署

## 技术特点
- 技术栈：Python3.9、PyQt5、OpenCV、Ultralytics、PyTorch1.9
- 部署方式：提供完整源码、UI界面、图标及运行教程
- 模型状态：预训练模型可直接使用，已配置成功环境
